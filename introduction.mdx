## How DNSHE compares

Most free subdomain services impose short renewal windows (sometimes just a few days before expiry), provide no API access, and support only basic record types. DNSHE takes a different approach:

| Feature | DNSHE | Typical free services |
| :--- | :--- | :--- |
| Renewal window | 180 days | 7–30 days |
| REST API | Yes — full management | None or read-only |
| Anycast infrastructure | Yes | No |
| Record types | A, AAAA, CNAME, MX, TXT, NS, SRV, CAA | A, CNAME only |
| Cost | Free, non-profit | Free (ad-supported) |

## Supported record types

DNSHE supports the full set of standard DNS record types:

| Type | Purpose |
| :--- | :--- |
| **A** | Maps a hostname to an IPv4 address |
| **AAAA** | Maps a hostname to an IPv6 address |
| **CNAME** | Aliases one hostname to another |
| **MX** | Routes email for a domain |
| **TXT** | Stores arbitrary text — used for SPF, DKIM, domain verification |
| **NS** | Delegates a subdomain to a different nameserver |
| **SRV** | Specifies a host and port for a service |
| **CAA** | Restricts which certificate authorities may issue TLS certificates |
